lifewatch data capture

Funktion för hämntning av data till lifewatch
date - datum för att bara söka ut "filer" som är uppdaterat efter datumet. Format: YYYY-MM-DD
page size - max antal poster i en fil. page - om inte allt får plats i en fil iterera up från 1 tills du får en tom fil. Filter för export: Country = "Sweden" Fil format? Character code = UTF8; brytning mellan poster= Windows style line endings \r\n; Brytning mellan fält = tab \t.
